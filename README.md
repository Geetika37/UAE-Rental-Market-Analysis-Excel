📊 UAE Residential Rental Market Dashboard – 2023

This project presents an Excel-based interactive dashboard analyzing residential rental market trends across the UAE for the year 2023.

The objective of this analysis is to uncover rental distribution patterns, seasonal trends, property segmentation, and market dominance across emirates using data-driven techniques.

**Dataset Summary**
Total Properties Analyzed: 14,233
Geographical Coverage: Major UAE emirates
Property Attributes: Rent, Property Type, Furnishing Status, Location, Monthly Trends

📈 **Dashboard Includes**
🔹KPI Cards (Total Listings, Average Rent, Distribution)
🔹Emirate-wise Rental Distribution
🔹Monthly Rent Trend Analysis
🔹Property Type Analysis
🔹Furnishing Status Breakdown
🔹Rental Segmentation (Percentile-based Classification)

🔍 **Key Insights**
🔹Dubai and Abu Dhabi dominate the rental listings, contributing the highest share of properties in the dataset.
🔹82% of properties are unfurnished, indicating strong long-term rental demand across the market.
🔹Rental distribution is concentrated in the Premium and Affordable segments, showing a polarized pricing structure with a smaller mid-range presence.
🔹A sharp rent increase is observed from August onwards, suggesting seasonal demand pressure during the latter part of the year.
🔹Apartments dominate the listings, reflecting high urban and city-based housing demand.

🛠 **Tools Used Microsoft Excel**
Pivot Tables Data Visualization Dashboard Design Slicers & Filters

📊 **Analytical Functions**
PERCENTILE.INC
  Used to classify rental prices into:
  Affordable
  Mid-Range
  Premium
TEXTBEFORE
  Used to extract property names from address fields for better categorization.

✨ **Project Objective**
This project analyzes 14,233 UAE residential rental listings for 2023 using Excel to uncover pricing trends, market segmentation, seasonal patterns, and emirate-wise rental dominance.
